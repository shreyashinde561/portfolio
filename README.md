# Shreya Shinde Portfolio
A personal portfolio website showcasing my web development projects, skills, and experience. Built using HTML, CSS, Python, Flask, and MERN stack projects.

---

## Features
- Responsive portfolio design
- Projects showcase with descriptions
- Contact form for direct communication
- Skills, Education, and Experience sections

---

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Backend:** Python, Flask, Node.js, Express.js
- **Database:** MySQL, MongoDB
- **Version Control:** Git & GitHub
- **Other:** AI/ML models for crop disease prediction
